<svg> - Contenedor SVG

Descripción: Define un lienzo SVG en el que se dibujan formas.

Description: Defines an SVG canvas where shapes are drawn.

Ejemplo

<svg width="200" height="100">
<!-- contenido gráfico aquí -->
</svg>

<rect> - Rectángulo

Descripción: Dibuja un rectángulo dentro del lienzo SVG.

Description: Draws a rectangle within the SVG canvas.

Ejemplo

<svg width="200" height="100">
    <rect x="10" y="10" width="150" height="80" fill="blue" />
</svg>

<circle> - Círculo

Descripción: Dibuja un círculo.

Description: Draws a circle.

Ejemplo

<svg width="200" height="100">
    <circle cx="50" cy="50" r="40" fill="red" />
</svg>

<line> - Línea

Descripción: Dibuja una línea recta entre dos puntos.

Description: Draws a straight line between two points.

Ejemplo

<svg width="200" height="100">
    <line x1="10" y1="10" x2="180" y2="90" stroke="black" stroke-width="3" />
</svg>

<text> - Texto SVG

Descripción: Inserta texto dentro del SVG.

Description: Inserts text within an SVG canvas.

Ejemplo

<svg width="200" height="100">
    <text x="20" y="50" font-size="24" fill="green">Hola</text>
